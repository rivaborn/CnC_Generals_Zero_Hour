# Generals/Code/Libraries/Source/WWVegas/WWMath/wwmathids.h

## Purpose
Defines chunk IDs for WWMath serialization, used in save/load operations.

## Responsibilities
- Declares enum values for 1D and 3D curve/spline types
- Assigns unique IDs for each math object type
- Provides constants for persist factory ID management
