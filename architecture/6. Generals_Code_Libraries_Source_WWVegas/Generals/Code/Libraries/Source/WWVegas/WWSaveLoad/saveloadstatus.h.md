# Generals/Code/Libraries/Source/WWVegas/WWSaveLoad/saveloadstatus.h

## Purpose
Header file defining the SaveLoadStatus namespace for managing save/load operation status text.

## Responsibilities
- Declares functions to set and retrieve status text for save/load operations.
- Provides macros for initializing primary and secondary status text.
- Uses WWString for string handling.

## Key Types
- None

## Key Functions
### Set_Status_Text
- Purpose: Sets the status text for a given ID (0 or 1).
- Calls: None

### Get_Status_Text
- Purpose: Retrieves the status text for a given ID.
- Calls: None

## Globals
- None

## Dependencies
- "always.h" (for basic includes)
- "wwstring.h" (for StringClass)
- Uses namespace SaveLoadStatus
