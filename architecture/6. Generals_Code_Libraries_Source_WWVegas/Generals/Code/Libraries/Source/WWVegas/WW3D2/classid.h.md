# Generals/Code/Libraries/Source/WWVegas/WW3D2/classid.h

## Purpose
Defines class IDs for WW3D (Westwood 3D) engine components, used for runtime type identification.

## Responsibilities
- Declares an enum of WW3D class IDs
- Assigns unique hex values to each class type
- Provides identifiers for texture, model, and point group classes

## Key Types
- (anonymous enum) (Enum): Contains all WW3D class IDs with associated hex values
