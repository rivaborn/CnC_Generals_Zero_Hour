# Generals/Code/Libraries/Source/WWVegas/WWLib/BSEARCH.H

## Purpose
Provides a templated binary search function optimized for inlined comparison operations.

## Responsibilities
- Implements binary search algorithm for sorted arrays
- Optimizes for cases where comparison is expensive
- Returns pointer to found element or NULL if not found

## Key Types
- None

## Key Functions
### Binary_Search
- Purpose: Performs binary search on a sorted array to find a target value
- Calls: None (pure algorithmic implementation)

## Globals
- None

## Dependencies
- Standard C++ template mechanisms
- Basic arithmetic and pointer operations
