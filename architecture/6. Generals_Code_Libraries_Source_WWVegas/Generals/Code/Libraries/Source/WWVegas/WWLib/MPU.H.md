# Generals/Code/Libraries/Source/WWVegas/WWLib/MPU.H

## Purpose
Header file for CPU-related utility functions, providing access to CPU clock speed and rate.

## Responsibilities
- Declares CPU clock speed retrieval functions
- Provides RDTSC-based CPU speed measurement
- Exposes CPU rate querying interface

- Purpose: Gets CPU rate information

- Calls: Not inferable
