# Generals/Code/Libraries/Source/WWVegas/WWLib/trackxy.h

## Purpose
Header file defining a simple `TrackXY` class for tracking 2D coordinates.

## Responsibilities
- Defines a lightweight coordinate tracking class
- Provides getter/setter methods for X/Y values
- Intended for use by the Surface class (or similar)

## Key Types
- TrackXY (class): Tracks current XY location with basic accessors

## Key Functions
### TrackXY::Set
- Purpose: Sets the current X/Y coordinates
- Calls: None

### TrackXY::Get_X
- Purpose: Returns the current X coordinate
- Calls: None

### TrackXY::Get_Y
- Purpose: Returns the current Y coordinate
- Calls: None

## Globals
- None

## Dependencies
- None (header-only, no includes)
