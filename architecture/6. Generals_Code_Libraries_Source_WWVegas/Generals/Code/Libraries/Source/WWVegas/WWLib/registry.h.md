# Generals/Code/Libraries/Source/WWVegas/WWLib/registry.h

## Purpose
Header file defining the `RegistryClass` for Windows registry access in the SAGE engine.

## Responsibilities
- Provides interface for reading/writing registry values (int, bool, float, string, binary)
- Supports value enumeration and deletion
- Manages registry key validity state

## Key Types
- **RegistryClass (Class)**: Wrapper for Windows registry operations.

## Key Functions
### RegistryClass::Get_Int
- Purpose: Retrieves an integer value from the registry.
- Calls: Not inferable.

### RegistryClass::Set_Int
- Purpose: Writes an integer value to the registry.
- Calls: Not inferable.

### RegistryClass::Get_Bool
- Purpose: Retrieves a boolean value from the registry.
- Calls: Not inferable.

### RegistryClass::Set_Bool
- Purpose: Writes a boolean value to the registry.
- Calls: Not inferable.

### RegistryClass::Get_Float
- Purpose: Retrieves a float value from the registry.
- Calls: Not inferable.

### RegistryClass::Set_Float
- Purpose: Writes a float value to the registry.
- Calls: Not inferable.

### RegistryClass::Get_String
- Purpose: Retrieves a string value from the registry (multiple overloads).
- Calls: Not inferable.

### RegistryClass::Set_String
- Purpose: Writes a string value to the registry (multiple overloads).
- Calls: Not inferable.

### RegistryClass::Get_Bin
- Purpose: Retrieves binary data from the registry.
- Calls: Not inferable.

### RegistryClass::Set_Bin
- Purpose: Writes binary data to the registry.
- Calls: Not inferable.
