# Generals/Code/Libraries/Source/WWVegas/WWLib/wwfont.h

## Purpose
Defines the `WWFontClass` for handling font data generated by legacy `FONTMAKE.EXE` utility.

## Responsibilities
- Manages font rendering with optional outlines/shadows
- Handles font spacing and remapping
- Provides pixel width/height calculations
- Integrates with `ConvertClass` for color conversion

## Key Types
- `WWFontClass` (Class): Concrete font class for legacy font data
- `FontType` (Struct): Header structure for font data file
- `ConvertClass` (Class): External converter for color remapping

## Key Functions
### `WWFontClass::Print`
- Purpose: Renders text on a surface with optional remapping
- Calls: `ConvertClass` methods, `Surface` methods

### `WWFontClass::Char_Pixel_Width`
- Purpose: Returns pixel width of a character
- Calls: None

### `WWFontClass::String_Pixel_Width`
- Purpose: Returns pixel width of a string
- Calls: `Char_Pixel_Width`

## Globals
- None

## Dependencies
- `font.h` (base `FontClass`)
- `surface.h` (for rendering)
- `ConvertClass` (external color converter)
