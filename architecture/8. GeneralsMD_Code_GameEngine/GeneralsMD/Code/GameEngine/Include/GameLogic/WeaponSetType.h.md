# GeneralsMD/Code/GameEngine/Include/GameLogic/WeaponSetType.h

## Purpose
Defines an enum for weapon set types used in the game, representing different weapon configurations for units.

## Responsibilities
- Declares the `WeaponSetType` enum with various weapon set flags.
- Documents the importance of bit shifting and state management.
- References external data structures (`TheWeaponSetNames`, `TheWeaponSetTypeToModelConditionTypeMap`) that must be updated when modifying the enum.

## Key Types
- **WeaponSetType (Enum)**: Represents different weapon configurations (e.g., veteran, elite, hero, upgrades, riders).
