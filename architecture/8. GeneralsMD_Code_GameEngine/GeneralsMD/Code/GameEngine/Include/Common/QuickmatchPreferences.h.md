# GeneralsMD/Code/GameEngine/Include/Common/QuickmatchPreferences.h

## Purpose
Defines the `QuickMatchPreferences` class for saving/loading quickmatch game settings.

## Responsibilities
- Manage quickmatch preferences (map selection, ladder settings, game limits).
- Provide getters/setters for preference values.
- Inherit from `UserPreferences` for base functionality.

## Key Types
- **QuickMatchPreferences (Class)**: Stores and manages quickmatch preferences.
