# GeneralsMD/Code/GameEngine/Include/Common/AudioHandleSpecialValues.h

## Purpose
Defines special values for audio handles in the game's audio system.

## Responsibilities
- Declares an enum for special audio handle values
- Provides constants for error states and special audio behaviors

## Key Types
- AudioHandleSpecialValues (Enum): Special values for audio handles
- AHSV_Error (Enum): Indicates an audio error
- AHSV_NoSound (Enum): Indicates no sound should play
- AHSV_Muted (Enum): Indicates sound is muted
- AHSV_NotForLocal (Enum): Indicates sound should not play locally
- AHSV_StopTheMusic (Enum): Indicates music should stop
- AHSV_StopTheMusicFade (Enum): Indicates music should fade out
- AHSV_FirstHandle (Enum): Marks the start of valid audio handles
