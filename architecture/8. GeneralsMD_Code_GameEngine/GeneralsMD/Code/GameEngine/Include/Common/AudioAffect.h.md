# GeneralsMD/Code/GameEngine/Include/Common/AudioAffect.h

## Purpose
Defines an enum for categorizing different types of audio affects in the game.

## Responsibilities
- Declares the `AudioAffect` enum for audio categorization.
- Provides bitmask flags for different audio types (music, sound, 3D sound, speech).
- Includes a combined flag for all audio types.
- Defines a system setting flag for volume adjustments.

## Key Types
- **AudioAffect (Enum)**: Flags for different audio categories (music, sound, 3D sound, speech, all, system setting).

None.

None.
