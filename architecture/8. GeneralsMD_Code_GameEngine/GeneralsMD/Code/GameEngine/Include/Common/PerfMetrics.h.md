# GeneralsMD/Code/GameEngine/Include/Common/PerfMetrics.h

## Purpose
Defines performance metric timing constants for frame-based measurements in the SAGE engine.

## Responsibilities
- Declares timing constants for load and logic frame metrics
- Defines spacing constant for metric display
- Provides enum values for performance measurement hooks

## Key Types
- PERFMETRICS_LOAD_STARTFRAME (Enum): Marks start of load timing
- PERFMETRICS_LOAD_STOPFRAME (Enum): Marks end of load timing
- PERFMETRICS_LOGIC_STARTFRAME (Enum): Marks start of logic timing
- PERFMETRICS_LOGIC_STOPFRAME (Enum): Marks end of logic timing
- PERFMETRICS_BETWEEN_METRICS (Enum): Spacing between metrics display

## Key Functions
None

## Globals
None

## Dependencies
- None (header-only, no includes)
