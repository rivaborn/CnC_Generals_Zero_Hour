# GeneralsMD/Code/GameEngine/Include/Common/SystemInfo.h

## Purpose
Defines a system-wide boolean flag indicating whether the game is running in Unicode mode.

## Responsibilities
- Exposes a global flag for Unicode system detection
- Provides a simple way to check system encoding at compile time

## Key Types
None

## Key Functions
None

## Globals
- TheSystemIsUnicode (Bool): Indicates whether the system is using Unicode encoding

## Dependencies
- None (header-only, no external dependencies visible)
