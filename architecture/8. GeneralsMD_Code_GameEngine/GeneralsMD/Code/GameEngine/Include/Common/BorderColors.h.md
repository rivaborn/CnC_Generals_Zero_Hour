# GeneralsMD/Code/GameEngine/Include/Common/BorderColors.h

## Purpose
Defines border color constants and their names for UI rendering.

## Responsibilities
- Declares `BorderColor` struct to associate color names with ARGB values
- Provides a lookup table of predefined border colors
- Exposes the count of available border colors

## Key Types
- **BorderColor (struct)**: Associates a color name string with an ARGB integer value

## Key Functions
None

## Globals
- **BORDER_COLORS (const BorderColor[])**: Array of predefined border colors with names
- **BORDER_COLORS_SIZE (const long)**: Number of entries in BORDER_COLORS array

## Dependencies
- None
