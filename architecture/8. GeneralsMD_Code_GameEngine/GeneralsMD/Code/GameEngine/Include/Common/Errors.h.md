# GeneralsMD/Code/GameEngine/Include/Common/Errors.h

## Purpose
Defines error codes used throughout the game engine for exception handling.

## Responsibilities
- Declares the `ErrorCode` enum with distinct error values.
- Provides a base value (`ERROR_BASE`) for error code uniqueness.
- Documents common failure modes (memory, arguments, files, etc.).
- Marks `ERROR_LAST` for future expansion tracking.

## Key Types
- **ErrorCode (Enum)**: Standardized error codes for exceptions (e.g., memory, file corruption).

## Key Functions
None.

## Globals
None.

## Dependencies
- None (header-only, no external symbols).
