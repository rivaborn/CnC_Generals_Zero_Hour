# GeneralsMD/Code/GameEngine/Include/GameNetwork/GameSpyOverlay.h

## Purpose
Header file defining GameSpy overlay UI functionality for Generals Zero Hour.

## Responsibilities
- Declares overlay types and management functions
- Provides message box utilities for GameSpy overlays
- Defines interface for opening/closing/toggling overlays

## Key Types
- GSOverlayType (Enum): overlay screen types (player info, map select, etc.)

## Key Functions
### ClearGSMessageBoxes
- Purpose: Remove any existing GameSpy message boxes
- Calls: Not inferable

### GSMessageBoxOk
- Purpose: Display an OK-only message box
- Calls: Not inferable

### GSMessageBoxOkCancel
- Purpose: Display an OK/Cancel message box
- Calls: Not inferable

### GSMessageBoxYesNo
- Purpose: Display a Yes/No message box
- Calls: Not inferable

### RaiseGSMessageBox
- Purpose: Bring message box to foreground
- Calls: Not inferable

### GameSpyOpenOverlay
- Purpose: Open a specific GameSpy overlay
- Calls: Not inferable

### GameSpyCloseOverlay
- Purpose: Close a specific GameSpy overlay
- Calls: Not inferable

### GameSpyCloseAllOverlays
- Purpose: Close all GameSpy overlays
- Calls: Not inferable

### GameSpyIsOverlayOpen
- Purpose: Check if an overlay is open
- Calls: Not inferable

### GameSpyToggleOverlay
- Purpose: Toggle visibility of an overlay
- Calls: Not inferable

### GameSpyUpdateOverlays
- Purpose: Update all GameSpy overlays
- Calls: Not inferable

### ReOpenPlayerInfo
