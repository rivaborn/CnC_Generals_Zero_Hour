# GeneralsMD/Code/GameEngine/Include/GameNetwork/GUIUtil.h

## Purpose
Header file for GUI utility functions used in the game's network setup and player configuration screens.

## Responsibilities
- Declares functions for populating combo boxes with game data
- Manages visibility of underlying GUI elements
- Controls slot list updates and accept controls for player setup
- Provides access to game window and game info classes

## Key Types
- GameWindow (Class): Represents a GUI window element
- GameInfo (Class): Contains game state and player information

## Key Functions
### ShowUnderlyingGUIElements
- Purpose: Shows or hides underlying GUI elements based on parameters
- Calls: Not inferable

### PopulateColorComboBox
- Purpose: Fills a combo box with available player colors
- Calls: Not inferable

### PopulatePlayerTemplateComboBox
- Purpose: Fills a combo box with available player templates
- Calls: Not inferable

### PopulateTeamComboBox
- Purpose: Fills a combo box with available teams
- Calls: Not inferable

### PopulateStartingCashComboBox
- Purpose: Fills a combo box with starting cash options
- Calls: Not inferable

### EnableSlotListUpdates
- Purpose: Enables or disables slot list updates
- Calls: Not inferable

### AreSlotListUpdatesEnabled
- Purpose: Checks if slot list updates are enabled
- Calls: Not inferable

### UpdateSlotList
- Purpose: Updates the slot list with current game state
- Calls: Not inferable

### EnableAcceptControls
- Purpose: Enables or disables accept controls for a player slot
- Calls: Not inferable
