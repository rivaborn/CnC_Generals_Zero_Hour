# GeneralsMD/Code/GameEngine/Include/GameClient/GadgetPushButton.h

## Purpose
Header file providing utility functions for manipulating push button gadgets in the game UI.

## Responsibilities
- Defines helper functions for push button state management
- Provides accessors for button images, colors, and borders
- Supports check-like button behavior
- Handles text, clock, and overlay image rendering

## Key Types
None

## Key Functions
### GadgetCheckLikeButtonSetVisualCheck
- Purpose: Sets the visual state of a check-like button
- Calls: Not inferable

### GadgetCheckLikeButtonIsChecked
- Purpose: Checks if a check-like button is in checked state
- Calls: Not inferable

### GadgetButtonEnableCheckLike
- Purpose: Enables/disables check-like behavior for a button
- Calls: Not inferable

### GadgetButtonSetText
- Purpose: Sets the text displayed on a button
- Calls: Not inferable

### GadgetButtonDrawClock
- Purpose: Draws a clock-style progress indicator on the button
- Calls: Not inferable

### GadgetButtonDrawInverseClock
- Purpose: Draws an inverse clock-style progress indicator
- Calls: Not inferable

### GadgetButtonDrawOverlayImage
- Purpose: Draws an overlay image on the button
- Calls: Not inferable

### GadgetButtonSetBorder
- Purpose: Sets the border color and visibility for a button
- Calls: Not inferable

### GadgetButtonSetData / GadgetButtonGetData
- Purpose: Associates/gets arbitrary data with a button
- Calls: Not inferable

### GadgetButtonSetAltSound
- Purpose: Sets an alternative sound for button interactions
- Calls: Not inferable
