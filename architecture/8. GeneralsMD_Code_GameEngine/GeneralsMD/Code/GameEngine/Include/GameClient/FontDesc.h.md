# GeneralsMD/Code/GameEngine/Include/GameClient/FontDesc.h

## Purpose
Defines a simple structure for holding font descriptions in the game engine.

- Holds font metadata (name, size, bold flag)
- Provides basic font description interface
- Used for font configuration in the game client

## Key Types
- FontDesc (struct): Holds font description data (name, size, bold flag)

## Key Functions
### FontDesc::FontDesc
- Purpose: Default constructor for FontDesc.
- Calls: None

## Globals
