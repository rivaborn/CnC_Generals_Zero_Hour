# GeneralsMD/Code/GameEngineDevice/Include/W3DDevice/GameClient/W3DCustomScene.h

## Purpose
Defines an enumeration for custom scene rendering pass modes in the W3D engine.

## Responsibilities
- Declares `CustomScenePassModes` enum for scene rendering control.
- Provides constants for default and alpha mask rendering passes.

## Key Types
- **CustomScenePassModes (Enum)**: Defines rendering pass modes for scenes.
- **SCENE_PASS_DEFAULT (Enum)**: Default rendering pass mode.
- **SCENE_PASS_ALPHA_MASK (Enum)**: Alpha mask rendering pass mode.

## Key Functions
None.

## Globals
None.

## Dependencies
- None (header-only, no includes or external symbols).
