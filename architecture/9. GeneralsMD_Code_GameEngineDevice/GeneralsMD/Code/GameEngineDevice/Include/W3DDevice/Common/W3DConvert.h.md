# GeneralsMD/Code/GameEngineDevice/Include/W3DDevice/Common/W3DConvert.h

## Purpose
Header file declaring functions for converting between logical screen coordinates and pixel screen coordinates in the W3D rendering system.

## Responsibilities
- Declares coordinate conversion functions
- Provides interface for screen space transformations
- Defines external function signatures for W3D coordinate systems

## Key Types
None

## Key Functions
### W3DLogicalScreenToPixelScreen
- Purpose: Converts logical screen coordinates to pixel screen coordinates
- Calls: Not inferable (external declaration)

### PixelScreenToW3DLogicalScreen
- Purpose: Converts pixel screen coordinates to logical screen coordinates
- Calls: Not inferable (external declaration)

## Globals
None

## Dependencies
- `Lib/BaseType.h` (for Real and Int types)
- External functions declared but not defined in this header
