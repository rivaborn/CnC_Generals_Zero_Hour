# GeneralsMD/Code/GameEngineDevice/Source/W3DDevice/Common/W3DConvert.cpp

## Purpose
Converts between W3D logical screen coordinates and pixel screen coordinates.

## Responsibilities
- Converts W3D logical coordinates to pixel screen coordinates
- Converts pixel screen coordinates to W3D logical coordinates
- Handles coordinate system transformations for rendering

## Key Types
None

## Key Functions
### W3DLogicalScreenToPixelScreen
- Purpose: Converts W3D logical coordinates to pixel screen coordinates
- Calls: None

### PixelScreenToW3DLogicalScreen
- Purpose: Converts pixel screen coordinates to W3D logical coordinates
- Calls: None

## Globals
None

## Dependencies
- W3DDevice/Common/W3DConvert.h
- Real (floating-point type)
- Int (integer type)
- REAL_TO_INT (macro for type conversion)
