# Generals/Code/GameEngineDevice/Include/W3DDevice/GameClient/W3DGadget.h

## Purpose
Header file declaring rendering functions for various UI gadgets in the W3D (W3D) device layer.

## Responsibilities
- Declares drawing functions for UI gadgets (buttons, sliders, etc.)
- Defines constants for gadget rendering (e.g., line width)
- Provides forward declarations for gadget-related types
