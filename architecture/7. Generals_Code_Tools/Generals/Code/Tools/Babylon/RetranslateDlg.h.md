# Generals/Code/Tools/Babylon/RetranslateDlg.h

## Purpose
Header file for a dialog class used in the Babylon tool to handle text retranslation.

## Responsibilities
- Define the `RetranslateDlg` class for managing retranslation UI.
- Declare member variables for old and new text.
- Define message handlers for dialog buttons (Retranslate, Skip, NoRetranslate, SkipAll).
- Include necessary headers and constants.

## Key Types
- **RetranslateDlg (Class)**: Dialog class for handling text retranslation.
- **(anonymous enum) (Enum)**: Contains `IDD` enum value for dialog ID.
- **IDD (Enum)**: Dialog resource ID.
