# Generals/Code/Tools/GUIEdit/Include/DialogProc.h

## Purpose
Header file declaring dialog procedure callbacks for the GUIEdit tool.

## Responsibilities
- Declares dialog procedure functions for GUIEdit tool
- Provides external linkage for dialog callbacks
- Organizes dialog-related code in one place

## Key Types
None

## Key Functions
### NewLayoutDialogProc
- Purpose: Dialog procedure for new layout creation.
- Calls: Not inferable

### GridSettingsDialogProc
- Purpose: Dialog procedure for grid settings configuration.
- Calls: Not inferable

### CallbackEditorDialogProc
- Purpose: Dialog procedure for callback editor functionality.
- Calls: Not inferable

## Globals
None

## Dependencies
- `<windows.h>`: For Windows API types
- `HWND`, `UINT`, `WPARAM`, `LPARAM`: Windows dialog procedure parameters
