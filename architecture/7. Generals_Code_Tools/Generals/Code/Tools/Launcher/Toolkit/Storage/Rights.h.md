# Generals/Code/Tools/Launcher/Toolkit/Storage/Rights.h

## Purpose
Defines access privilege enumerations for file or resource permissions.

## Responsibilities
- Declares an enum type for access rights
- Provides symbolic constants for read-only, write-only, and read-write permissions
- Serves as a header for inclusion in other modules needing access control

## Key Types
- ERights (Enum): access privilege type (read-only, write-only, read-write)

None
