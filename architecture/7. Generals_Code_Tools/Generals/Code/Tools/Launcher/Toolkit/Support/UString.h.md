# Generals/Code/Tools/Launcher/Toolkit/Support/UString.h

## Purpose
Unicode string management class for the SAGE engine toolkit.

## Responsibilities
- Manages Unicode strings with reference counting
- Provides string operations (concat, compare, trim, etc.)
- Supports both ANSI and Unicode character sets
- Implements string manipulation operators

## Key Types
- UString (Class): Unicode string wrapper with reference counting

## Key Functions
### UString()
- Purpose: Default constructor
- Calls: Not inferable

### Length()
- Purpose: Returns string length
- Calls: Not inferable

### Copy()
- Purpose: Copies string content
- Calls: Not inferable

### Concat()
- Purpose: Concatenates strings
- Calls: Not inferable

### Compare()
- Purpose: Compares strings
- Calls: Not inferable

### Find()
- Purpose: Finds character position
- Calls: Not inferable

### SubString()
- Purpose: Extracts substring
- Calls: Not inferable

### Trim()
- Purpose: Removes whitespace
- Calls: Not inferable

### ConvertToANSI()
- Purpose: Converts to ANSI encoding
- Calls: Not inferable

### AllocString()
- Purpose: Allocates string buffer
- Calls: Not inferable

## Globals
None

- RefCounted.h
