# Generals/Code/Tools/WorldBuilder/include/StdAfx.h

## Purpose
Precompiled header file for the WorldBuilder tool, including standard system and project-specific headers.

- Includes common system headers (Windows, MFC)
- Includes project-specific headers (Basetype.h)
