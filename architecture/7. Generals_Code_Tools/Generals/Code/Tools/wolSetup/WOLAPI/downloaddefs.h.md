# Generals/Code/Tools/wolSetup/WOLAPI/downloaddefs.h

## Purpose
Defines constants for download statuses, return codes, and error codes used in the WOL (Westwood Online) download API.

## Responsibilities
- Defines download status enum values
- Defines download return codes as HRESULT values
- Defines download event return codes
- Defines error codes for download events

## Key Types
None

## Key Functions
None

## Globals
None

## Dependencies
- Uses `S_OK` and `MAKE_HRESULT` macros (Windows HRESULT definitions)
- Uses `SEVERITY_ERROR` and `FACILITY_ITF` constants (Windows error facility definitions)
