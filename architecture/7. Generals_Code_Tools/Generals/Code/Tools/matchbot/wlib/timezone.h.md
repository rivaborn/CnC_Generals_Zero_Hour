# Generals/Code/Tools/matchbot/wlib/timezone.h

## Purpose
Provides convenience functions to retrieve current timezone information, including daylight savings adjustments.

## Responsibilities
- Retrieve timezone description and offset from GMT
- Return current timezone string (with DST)
- Return current timezone offset from GMT

## Key Types
None

## Key Functions
### GetTimezoneInfo
- Purpose: Fills in timezone description and offset from GMT
- Calls: Not inferable

### TimezoneString
- Purpose: Returns description of current timezone (including DST)
- Calls: Not inferable

### TimezoneOffset
- Purpose: Returns offset from GMT of current timezone
- Calls: Not inferable

## Globals
None

## Dependencies
- No includes or external symbols used
