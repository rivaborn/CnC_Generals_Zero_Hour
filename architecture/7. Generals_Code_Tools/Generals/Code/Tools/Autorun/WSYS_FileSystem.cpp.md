# Generals/Code/Tools/Autorun/WSYS_FileSystem.cpp

## Purpose
This file defines the global file system singleton for the WSYS library, providing centralized file access.

## Responsibilities
- Declares the global `TheFileSystem` singleton instance
- Provides documentation for the file system's purpose and usage
- Includes necessary headers for file system functionality

## Key Types
- None

## Key Functions
- None

## Globals
- TheFileSystem (FileSystem*): The global file system singleton instance used for all file access operations.

## Dependencies
- `wsys_FileSystem.h`: Main header for file system declarations
- `wsys_StdFileSystem.h`: Standard file system implementation header
