# Generals/Code/Tools/WW3D/pluglib/BITTYPE.H

## Purpose
Defines basic fixed-size integer and floating-point type aliases for cross-platform consistency.

## Responsibilities
- Provides standardized type definitions for 8/16/32-bit integers and floats
- Ensures portability across different compilers and architectures
- Supports both unsigned and signed variants

## Key Types
- uint8 (typedef): 8-bit unsigned integer
- uint16 (typedef): 16-bit unsigned integer
- uint32 (typedef): 32-bit unsigned integer
- sint8 (typedef): 8-bit signed integer
- sint16 (typedef): 16-bit signed integer
- sint32 (typedef): 32-bit signed integer
- float32 (typedef): 32-bit floating-point
- float64 (typedef): 64-bit floating-point

None

None

- None (header-only, no external symbols)
