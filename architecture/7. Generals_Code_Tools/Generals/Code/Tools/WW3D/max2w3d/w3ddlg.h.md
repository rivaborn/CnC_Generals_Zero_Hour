# Generals/Code/Tools/WW3D/max2w3d/w3ddlg.h

## Purpose
Defines the W3dOptionsDialogClass for handling export options dialog in the max2w3d tool.

## Responsibilities
- Manages export options UI dialog
- Handles user input for W3D export settings
- Controls enable/disable states of export options
- Retrieves configured export options
- Integrates with 3ds Max SDK

## Key Types
- W3dOptionsDialogClass (Class): Manages the export options dialog and user interactions.
