# GeneralsMD/Code/Libraries/Source/WWVegas/wwshade/shd7bumpspec_constants.h

## Purpose
Defines constants and macros for a bump specular with gloss map shader in the SAGE engine.

## Responsibilities
- Declares vertex shader macros for matrix indices
- Defines input register names for shader inputs
- Specifies register names for intermediate calculations
- Identifies texture stage for cubemap
- Comments out unused matrix constants
