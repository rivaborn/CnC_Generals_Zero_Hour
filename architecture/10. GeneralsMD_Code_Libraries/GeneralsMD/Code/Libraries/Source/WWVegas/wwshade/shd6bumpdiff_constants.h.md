# GeneralsMD/Code/Libraries/Source/WWVegas/wwshade/shd6bumpdiff_constants.h

## Purpose
Defines constants and macros for a bump-diffuse shader in the SAGE engine, specifically for vertex shader operations.

## Responsibilities
- Defines constant register IDs for world/view/projection matrices
- Specifies input vertex attributes (position, normal, diffuse, texture coords)
- Declares shader registers for lighting and world-space calculations
- Sets texture stage identifiers for cubemap sampling
