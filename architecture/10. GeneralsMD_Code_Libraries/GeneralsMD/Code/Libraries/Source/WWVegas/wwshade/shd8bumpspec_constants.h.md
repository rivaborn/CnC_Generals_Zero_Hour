# GeneralsMD/Code/Libraries/Source/WWVegas/wwshade/shd8bumpspec_constants.h

## Purpose
Defines constants and macros for DirectX 8 bump specular shader with gloss map.

## Responsibilities
- Defines vertex shader macros for matrix and lighting constants
- Defines pixel shader constants for texture stages and registers
- Specifies input registers for vertex shader
- Defines output and texture stage registers for pixel shader
