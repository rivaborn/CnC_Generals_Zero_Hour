# GeneralsMD/Code/Libraries/Source/WWVegas/wwshade/shd8bumpdiff_constants.h

## Purpose
Defines constants and macros for the DirectX 8 bump diffuse shader in the SAGE engine.

## Responsibilities
- Declares constant register IDs for vertex and pixel shaders
- Defines input register names for shader inputs
- Specifies texture stage names and color registers
- Provides macros for world/view/projection matrix constants
