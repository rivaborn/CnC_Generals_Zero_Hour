# GeneralsMD/Code/Libraries/Source/WWVegas/wwshade/shddeffactory.h

## Purpose
Defines factory classes for creating shader definitions in the SAGE engine.

## Responsibilities
- Provides base factory class for shader definitions
- Implements template-based factory for easy shader registration
- Defines macro for automatic shader registration
- Manages linked list of shader factories

## Key Types
- **ShdDefClass (Class)**: Base class for shader definitions (forward declared)
- **ShdDefFactoryClass (Class)**: Abstract factory for creating shader definitions
- **SimpleShdDefFactoryClass (Class)**: Template factory implementation for shader definitions

## Key Functions
### ShdDefFactoryClass::Create
- Purpose: Pure virtual method to create shader definition instances
- Calls: None

### ShdDefFactoryClass::Get_Name
- Purpose: Pure virtual method to get shader definition name
- Calls: None

### ShdDefFactoryClass::Get_Class_ID
- Purpose: Pure virtual method to get shader definition class ID
- Calls: None

### SimpleShdDefFactoryClass::Create
- Purpose: Creates new instance of template shader definition
- Calls: `new T`

### SimpleShdDefFactoryClass::Get_Name
- Purpose: Returns registered name of shader definition
- Calls: None

### SimpleShdDefFactoryClass::Get_Class_ID
- Purpose: Returns registered class ID of shader definition
- Calls: None

## Globals
- **T##Name (char[])**: Storage for shader definition name (generated by macro)
- **T##Factory (SimpleShdDefFactoryClass)**: Factory instance (generated by macro)

## Dependencies
- `always.h`: For basic types and macros
- `bittype.h`: For uint32 definition
- ShdDefManagerClass: Uses factory linked list
