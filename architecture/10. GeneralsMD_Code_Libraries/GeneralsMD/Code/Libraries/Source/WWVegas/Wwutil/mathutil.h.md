# GeneralsMD/Code/Libraries/Source/WWVegas/Wwutil/mathutil.h

## Purpose
Provides utility functions for mathematical operations, including vector/angle conversions, distance calculations, rounding, vector rotation, and random number generation.

## Responsibilities
- Vector/angle conversions
- Distance calculations
- Rounding operations
- Vector rotation
- Random number generation with various PDFs

## Key Types
- cMathUtil (Class): Static utility class for mathematical operations

## Key Functions
### Angle_To_Vector
- Purpose: Convert an angle to a direction vector
- Calls: Not inferable

### Vector_To_Angle
- Purpose: Convert a direction vector to an angle
- Calls: Not inferable

### Simple_Distance
- Purpose: Calculate Euclidean distance between two points
- Calls: Not inferable

### Round
- Purpose: Round a double to nearest integer
- Calls: Not inferable

### Rotate_Vector
- Purpose: Rotate a 2D vector by a given angle
- Calls: Not inferable

### Get_Uniform_Pdf_Double

- Calls: Not inferable

### Get_Normalized_Uniform_Pdf_Double

- Calls: Not inferable
