# GeneralsMD/Code/Libraries/Source/WWVegas/WWMath/culltype.h

## Purpose
Defines an enumeration for culling operation results used across WWMath, WW3D, and WWPhys modules.

## Responsibilities
- Declares the `CULLTYPE` enum for consistent culling result communication.
- Provides three culling states: outside, intersecting, and inside.
- Ensures type consistency for culling functions across the engine.

## Key Types
- CULLTYPE (Enum): Represents the result of a culling operation (outside, intersecting, or inside).

None.
