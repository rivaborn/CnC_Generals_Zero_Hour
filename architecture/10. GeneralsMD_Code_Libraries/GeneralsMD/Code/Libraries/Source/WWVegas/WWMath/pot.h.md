# GeneralsMD/Code/Libraries/Source/WWVegas/WWMath/pot.h

## Purpose
Header file declaring functions to find the next power of two (POT) for integer values.

## Responsibilities
- Declares `Find_POT` for signed integers.
- Declares `Find_POT_Log2` for unsigned integers.
- Provides header guards for inclusion safety.

## Key Types
- None

## Key Functions
### Find_POT
- Purpose: Returns the smallest power of two greater than or equal to the input value.
- Calls: Not inferable (implementation in separate file).

### Find_POT_Log2
- Purpose: Returns the smallest power of two greater than or equal to the input value, as a log2 result.
- Calls: Not inferable (implementation in separate file).

## Globals
- None

## Dependencies
- None (pure header file).
