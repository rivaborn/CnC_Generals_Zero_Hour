# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/wwfont.h

## Purpose
Defines the `WWFontClass`, a concrete font class for handling font data generated by legacy FONTMAKE.EXE utility.

## Responsibilities
- Manage font data, including outlined and shadowed fonts
- Provide text rendering and measurement capabilities
- Support font palette remapping and color conversion
- Handle font spacing adjustments

## Key Types
- `WWFontClass` (Class): Concrete font class for legacy font data
- `FontType` (Struct): Header structure for font data file
- `ConvertClass` (Class): External class for color conversion (forward declared)

## Key Functions
### `WWFontClass::Print`
- Purpose: Renders text on a surface with optional clipping and color conversion
- Calls: Not inferable (implementation in .cpp)

### `WWFontClass::Char_Pixel_Width`
- Purpose: Returns pixel width of a single character
- Calls: Not inferable

### `WWFontClass::String_Pixel_Width`
- Purpose: Returns pixel width of a string
- Calls: Not inferable

## Globals
- None

## Dependencies
- `font.h` (base class)
- `surface.h` (for rendering)
- `ConvertClass` (external color conversion)
