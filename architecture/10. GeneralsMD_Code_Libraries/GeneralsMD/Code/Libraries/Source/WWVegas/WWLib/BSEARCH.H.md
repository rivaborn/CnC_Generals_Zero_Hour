# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/BSEARCH.H

## Purpose
Provides a templated binary search function optimized for inlined comparison operations.

## Responsibilities
- Implements a binary search algorithm for sorted arrays.
- Optimizes for cases where comparison is expensive by inlining.
- Returns a pointer to the found element or NULL if not found.

## Key Functions
### Binary_Search
- Purpose: Performs binary search on a sorted array to find a target value.
- Calls: None (pure algorithmic implementation)
