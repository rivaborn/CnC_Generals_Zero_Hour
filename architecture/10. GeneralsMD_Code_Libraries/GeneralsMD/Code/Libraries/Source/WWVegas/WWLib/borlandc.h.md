# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/borlandc.h

## Purpose
Header file for Borland C++ compiler compatibility in the SAGE engine.

## Responsibilities
- Defines `BORLANDC_H` guard when using Borland C++ compiler
- Notes Borland C++'s compliance with C++ standards
- Includes conditional compilation for MSVC

## Key Types
None

## Key Functions
None

## Globals
None

## Dependencies
- `__BORLANDC__` (macro): Borland C++ compiler detection
- `_MSC_VER` (macro): MSVC compiler version detection
