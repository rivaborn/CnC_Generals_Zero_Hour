# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/incdec.h

## Purpose
Provides template operators for incrementing and decrementing enumeration types.

## Responsibilities
- Defines `operator++` and `operator--` for generic types.
- Enables bitwise arithmetic on enumerations.
- Supports both prefix and postfix forms.

## Key Types
None

## Key Functions
### `operator++(T & a)`
- Purpose: Prefix increment operator for type T.
- Calls: None

### `operator++(T & a, int)`
- Purpose: Postfix increment operator for type T.
- Calls: None

### `operator--(T & a)`
- Purpose: Prefix decrement operator for type T.
- Calls: None

### `operator--(T & a, int)`
- Purpose: Postfix decrement operator for type T.
- Calls: None

## Globals
None

## Dependencies
