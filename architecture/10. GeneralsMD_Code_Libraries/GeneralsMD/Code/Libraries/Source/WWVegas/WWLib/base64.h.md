# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/base64.h

## Purpose
Header file declaring Base64 encoding/decoding functions for the SAGE engine.

## Responsibilities
- Declares Base64 encoding and decoding function prototypes
- Provides interface for binary-to-text and text-to-binary conversion
- Supports data length parameters for input/output buffers

## Key Types
None

## Key Functions
### Base64_Encode
- Purpose: Encodes binary data to Base64 text representation
- Calls: Not inferable

### Base64_Decode
- Purpose: Decodes Base64 text back to binary data
- Calls: Not inferable

## Globals
None

## Dependencies
- Standard C/C++ types (void const*, int)
- No external symbols used
