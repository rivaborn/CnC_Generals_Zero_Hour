# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/MPU.H

## Purpose
Header file for CPU-related utility functions, providing access to CPU clock speed and rate.

## Responsibilities
- Declares functions to retrieve CPU clock speed and rate.
- Provides interface for CPU performance measurement.
- Exposes RDTSC-based CPU speed detection.

## Key Types
- None

## Key Functions
### Get_CPU_Clock
- Purpose: Retrieves the CPU clock frequency.
- Calls: None

### Get_CPU_Rate
- Purpose: Retrieves the CPU rate (clock speed).
- Calls: None

### Get_RDTSC_CPU_Speed
- Purpose: Detects CPU speed using RDTSC instruction.
- Calls: None

## Globals
- None

## Dependencies
- None (header-only file)
