# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/gcd_lcm.h

## Purpose
Header file declaring functions for computing greatest common divisor (GCD) and least common multiple (LCM).

## Responsibilities
- Declares GCD and LCM calculation functions
- Provides mathematical utilities for integer operations
- Supports modular arithmetic needs in the engine

## Key Types
None

## Key Functions
### Greatest_Common_Divisor
- Purpose: Computes the greatest common divisor of two unsigned integers
- Calls: Not inferable

### Least_Common_Multiple
- Purpose: Computes the least common multiple of two unsigned integers
- Calls: Not inferable

## Globals
None

## Dependencies
- "always.h": Header include for common definitions
- Uses unsigned int type from standard library
