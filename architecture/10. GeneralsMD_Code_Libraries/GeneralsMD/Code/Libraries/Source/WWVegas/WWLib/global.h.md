# GeneralsMD/Code/Libraries/Source/WWVegas/WWLib/global.h

## Purpose
Defines basic types and macros for the WWLib library, including pointer and integer types.

## Responsibilities
- Defines generic pointer type (POINTER)
- Defines fixed-size integer types (UINT2, UINT4)
- Provides PROTO_LIST macro for function prototyping
- Sets default value for PROTOTYPES flag

## Key Types
- POINTER (Class): generic pointer type (unsigned char*)
- UINT2 (Class): 2-byte unsigned integer type
- UINT4 (Class): 4-byte unsigned integer type

## Key Functions
None

## Globals
None

## Dependencies
- None (header-only file)
