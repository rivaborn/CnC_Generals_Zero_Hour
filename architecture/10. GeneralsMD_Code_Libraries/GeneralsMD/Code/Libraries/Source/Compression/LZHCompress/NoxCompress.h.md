# GeneralsMD/Code/Libraries/Source/Compression/LZHCompress/NoxCompress.h

## Purpose
Header file declaring compression/decompression functions for file and memory operations.

## Responsibilities
- Declares file-based compression/decompression functions
- Declares memory-based compression/decompression functions
- Defines constants for file extensions
- Provides size calculation for compressed data

## Key Types
None

## Key Functions
### DecompressFile
- Purpose: Decompress a file from input to output path
- Calls: Not inferable

### CompressFile
- Purpose: Compress a file from input to output path
- Calls: Not inferable

### CompressPacket
- Purpose: Compress a packet of data
- Calls: Not inferable

### DecompressPacket
- Purpose: Decompress a packet of data
- Calls: Not inferable

### CalcNewSize
- Purpose: Calculate the new size after compression
- Calls: Not inferable

### DecompressMemory
- Purpose: Decompress memory buffer
- Calls: Not inferable

### CompressMemory
- Purpose: Compress memory buffer
- Calls: Not inferable

## Globals
- MAP_EXTENSION (const char*): Extension for map files
- LZH_EXTENSION (const char*): Extension for compressed files
- RUL_EXT
