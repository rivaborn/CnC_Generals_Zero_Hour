# GeneralsMD/Code/Libraries/Include/Lib/trig.h

## Purpose
Header file declaring fast trigonometric functions for the SAGE engine.

## Responsibilities
- Declares basic trigonometric functions (sin, cos, tan, acos, asin)
- Provides fast implementations for game math operations
- Exposes Real-type trig functions for engine use

## Key Types
None

## Key Functions
### Sin(Real)
- Purpose: Computes sine of an angle in radians.
- Calls: Not inferable

### Cos(Real)
- Purpose: Computes cosine of an angle in radians.
- Calls: Not inferable

### Tan(Real)
- Purpose: Computes tangent of an angle in radians.
- Calls: Not inferable

### ACos(Real)
- Purpose: Computes arccosine of a value.
- Calls: Not inferable

### ASin(Real)
- Purpose: Computes arcsine of a value.
- Calls: Not inferable
